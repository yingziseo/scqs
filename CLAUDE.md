# SCQS - SEO文章批量生成工具

## 项目概述

SEO文章批量生成Web工具，支持5步向导式操作流程：
1. 输入关键词 → 2. 选择标题 → 3. 生成文章 → 4. 编辑内容 → 5. 生成配图并导出

## 技术栈

- **前端**: Vite + React + TypeScript + Zustand
- **后端**: Express.js + better-sqlite3
- **AI接口**:
  - 智谱 GLM-4 (文本生成)
  - 豆包 Doubao Seedream 4.0 (图片生成)

## 目录结构

```
scqs/
├── client/          # 前端 React 应用
│   └── src/
│       ├── App.tsx  # 主应用 (5步向导)
│       └── store/   # Zustand 状态管理
├── server/          # 后端 Express 服务
│   └── src/
│       ├── app.js           # 入口
│       ├── routes/          # API 路由
│       ├── services/        # AI 服务封装

│       │   └── doubao.js    # 豆包文生图
│       └── db/              # SQLite 数据库
└── CLAUDE.md        # 开发日志
```

## API 配置

- `GLM_API_KEY` - 智谱API密钥
- `DOUBAO_API_KEY` - 豆包API密钥
- `DOUBAO_MODEL` - 豆包模型名 (当前: doubao-seedream-4-0-250828)
- `PORT` - 服务端口 (默认: 3001)

## 图片生成配置

- **尺寸**: 2048x2048 (2K 1:1 正方形)
- **水印**: 强制添加 (火山引擎限制，需商业授权去除)
